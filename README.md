awesomewm-rc
============

###usage:
- make a backup of your rc.lua
- download repository
- place contents in ~/.config/awesome/
- change theme to whatever you want to use in rc.lua
- restart awesome

###dependencies:
- acpiclient
- alsa-utils
- compton
- lain-git
- network-manager-applet
- networkmanager
- sp (command-line client for spotify)
- vicious
